# Моторное масло

Для полной заливки #re-classic-350-reborn на сухую требуется 2.2 литра, при замене - 1.7 литра. См. [[../owners-manual-re-classic-350-reborn.pdf|owners-manual-re-classic-350-reborn]].

## Вариации

Для #re-classic-350-reborn и других RE с мотором #J-серия рекомендуют следующие масла (15W-50):

- Для повседневной эксплуатации (микс город + шоссе):
	- От производителя: [Royal Enfield Liquid Gun Engine Oil](https://www.amazon.in/Royal-Enfield-Liquid-Engine-Synthetic-15W50-1L/dp/B0CDWDTFK1) (SAE 15W-50 API SL Grade JASO MA2 Semi Synthetic).
	- [Shell Advance](https://www.ozon.ru/product/shell-advance-15w-50-maslo-motornoe-polusinteticheskoe-1-l-248068899/?at=LZtlnnJ3LIoxQByBtwnPN8xtJQm57lTyPk18JtjGykQ1&tab=reviews).
	- [Motul 5100 4T](https://www.ozon.ru/product/motul-5100-4t-15w-50-maslo-motornoe-sinteticheskoe-1-l-1095469581/?at=nRtrNNJWncJlJNvQCZx2D5wI25KlxmcoDR9RnH8o1p7k).
- Для дальних поездок (шоссе):
	- [Liqui Moly Motorbike 4T Street](https://www.ozon.ru/product/liqui-moly-motorbike-4t-street-15w-50-maslo-motornoe-sinteticheskoe-1-l-189151047/?at=k2tozzMwghQXpyK5un25Rl6izvZg9kU8DND4ZT66q3AL).
	- [Motul 300v 4T](https://www.ozon.ru/product/motul-300v-15w-50-maslo-motornoe-sinteticheskoe-1-l-1678545410/?at=z6tOoo0rKc2ROOWwTqJLQLnix6KLpJSnBnpzxUzLQMJl).

Также можно 10W-50 или 10W-40, в зависимости от условий эксплуатации.
