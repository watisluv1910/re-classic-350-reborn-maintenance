# Руководство к обслуживанию

В данном модуле представлены пошаговые инструкции по самостоятельному обслуживанию мотоцикла [Royal Enfield](https://royalenfieldrussia.com/) #classic-350-reborn в частности и других мотоциклов с двигателем #J-серия в целом.

## Разделы

Руководство содержит следующие разделы:

1. [[Инструменты/index|Инструменты]] – в нём собрана информация о внешнем виде, свойствах и местах возможной покупки большинства [[Инструменты/index#Определение инструмента|инструментов]], которые могут пригодиться при базовом обслуживании.
2. [[Обслуживание/index|Обслуживание]] - в нём собраны подробные пошаговые инструкции по выполнению операций по самостоятельному обслуживанию #classic-350-reborn. Инструкции были подготовлены на основе сборника [All New Classic 350 DIY Videos](https://youtube.com/playlist?list=PLT1BEDU54E1QNhNm0RBd0BGXzAH1leldc&si=_gbwqCGGBN8w6IeR) с официального [канала Royal Enfield](https://www.youtube.com/@royalenfield) на YouTube. В качестве дополнительной информации в инструкциях приводятся сведения из [[owners-manual-re-classic-350-reborn.pdf|мануала]] и из различных форумов владельцев RE.
3. [[Расходники/index|Расходники]] – в нём собрана информация о внешнем виде, свойствах и местах возможной покупки большинства [[Расходники/index#Определение расходника|расходников]], которые могут пригодиться при базовом обслуживании.

- #J-серия – заметка актуальна для всех мотоциклов [Royal Enfield на J-платформе](https://motorcycletouring.quora.com/What-is-the-J-platform-in-Royal-Enfield-How-is-it-different-from-the-previous-ones). На момент написания: Meteor 350, Classic 350, Hunter 350, Bullet 350 и Goan Classic 350.
- #bullet-350-reborn – заметка актуальна для [Bullet 350](https://www.royalenfield.com/us/en/motorcycles/bullet-350).
- #classic-350-reborn – заметка актуальна для [Classic 350](https://www.royalenfield.com/us/en/motorcycles/classic-350).
- #meteor – заметка актуальна для [Meteor 350](https://www.royalenfield.com/us/en/motorcycles/meteor).
- #hunter – заметка актуальна для [Hunter 350](https://www.royalenfield.com/us/en/motorcycles/hunter-350).
- #goan – заметка актуальна для [Goan Classic 350](https://www.royalenfield.com/us/en/motorcycles/goan-classic-350).
