# Заправка мотоцикла

## Требуемые инструменты

Может понадобиться:
- [[../Инструменты/Воронка для технических жидкостей|Воронка для технических жидкостей]].

## Порядок действий

1. [[Выключение зажигания|Выключить зажигание]] и вынуть ключ.
2. Открыть крышку бензобака:
   ![[attachments/fuel-tank-cap.png]]
3. Залить АИ-92 или АИ-95 бензин в горловину с помощью воронки или топливного пистолета.
4. Закрыть крышку бензобака, надовив на неё для фиксации. Вытянуть ключ.
