# Проверка и замена предохранителей

## Причины проведения

- Периодическая проверка.
- Мотоцикл не заводится.
- Не работает лампа головного света или другая светотехника.
- Не работает АБС.
- Не работает гудок.

Иные проблемы с электроникой.

## Требуемые инструменты

- [[../Инструменты/Зажим для извлечения предохранителей|Зажим для извлечения предохранителей]].

## Требуемые расходники

- [[../Расходники/Предохранитель|Предохранитель]].

## Порядок действий

1. [[Выключение зажигания|Выключить зажигание]] и вынуть ключ.
2. Открыть правую [[Открытие боковой панели|боковую панель]].
3. Открыть коробку с предохранителями (1), нажав на защёлку (2) и подняв крышку (3):
   ![[attachments/fuse-box.png| center | 400]]
4. После открытия крышки коробки для каждого из 6 действующих предохранителей по очереди:
	1. Вынуть с помощью [[../Инструменты/Зажим для извлечения предохранителей|зажима]].
	2. Проверить провод внутри предохранителя:
	   ![[attachments/Pasted image 20260109225355.png]]
	   Если провод оборван (2) – заменить идентичным предохранителем из запасных:
	   ![[attachments/Pasted image 20260109230354.png]]
	3. Вставить оригинальный или замену в гнездо, слегка прижав пальцем для фиксации.
5. Закрыть крышку коробки, зафиксировав защёлку.
6. Повторить действия 3-5 для коробки предохранителей АБС (правее).
7. Закрыть правую боковую панель.

Если была произведена замена, восстановить использованный запасной предохранитель.
